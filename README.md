# Betting Service

## Introduction

This betting platform is a full-stack application designed to cater to betting enthusiasts, providing a comprehensive environment for placing bets, checking leaderboards, and viewing detailed game analytics. Built with Node.js and Express for the backend, a React.js with TypeScript frontend, and PostgreSQL for database management, it offers a robust solution for user authentication, bet management, and real-time leaderboard updates.

## Technology Stack

- **Backend**: Node.js, Express
- **Frontend**: React.js with TypeScript
- **Database**: PostgreSQL
- **Authentication**: bcrypt for hashing and secure password management
- **Styling**: Tailwind CSS (assumed for frontend styling)
- **Environment Management**: dotenv for local environment variables

## Features

- **User Authentication**: Register or log in to access the betting features.
- **Place Bets**: Navigate to the bet placing section to make a bet on available games.
- **Leaderboard**: Check the leaderboard to see top players based on their accumulated points.
- **Detailed Leaderboard View**: For more detailed analytics, visit the detailed leaderboard section.
- **Admin Panel**: Authorized users can update game data, manage bets, and adjust user points.
